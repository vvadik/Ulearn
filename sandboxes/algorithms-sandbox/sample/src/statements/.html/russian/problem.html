Вам заданы два целых числа $$$a$$$ и $$$b$$$. Выведите $$$a+b$$$.

Входные данные

Единственная строка входных данных содержит два целых числа $$$a$$$ и $$$b$$$ ($$$-100 \le a,b \le 100$$$).

Выходные данные

Выведите $$$a+b$$$.

Примеры

Входные данные
7 8
Выходные данные
15
Входные данные
-100 100
Выходные данные
0
Входные данные
-7 -99
Выходные данные
-106

Примечание

В первом примере $$$a=7$$$ и $$$b=8$$$. Таким образом, ответ равен $$$a+b=7+8=15$$$.
